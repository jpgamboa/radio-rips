Settings

Download Directory

Absolute path where ripped files are saved.

Audio Format

Output format for YouTube, Spotify, Apple Music, and Tidal downloads. "Best (no re-encode)" keeps the original codec from YouTube for maximum quality.

Spotify API

Required to download from Spotify URLs. Spotify tracks are resolved to YouTube and downloaded via yt-dlp. To get your credentials:

  1. Go to the Spotify Developer Dashboard
  2. Log in and click Create App
  3. Name it anything (e.g. "Radio Rips"), set redirect URI to http://localhost
  4. Copy the Client ID and Client Secret from the app settings

Deezer

Required for downloads from Deezer. To get your ARL:

  1. Log in to deezer.com
  2. Open browser DevTools (F12)
  3. Go to Application → Cookies → deezer.com
  4. Copy the value of the arl cookie

Apple Music & Tidal
