Settings

Download Directory

Absolute path where ripped files are saved.

Spotify API

Required to download from Spotify URLs. Spotify tracks are resolved to YouTube and downloaded via yt-dlp. To get your credentials:

  1. Go to the Spotify Developer Dashboard
  2. Log in and click Create App
  3. Name it anything (e.g. "Radio Rips"), set redirect URI to http://localhost
  4. Copy the Client ID and Client Secret from the app settings

Apple Music & Tidal

No setup needed. Paste any public Apple Music or Tidal link (track, album, or playlist) and tracks are automatically resolved to YouTube and downloaded via yt-dlp.
